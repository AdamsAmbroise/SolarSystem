# Simulation du Système Solaire - Instructions

## Lancer la simulation
Pour exécuter la simulation dans votre navigateur web (Chromium, Brave, Google Chrome, etc.) :

1. Ouvrez un terminal.
2. Lancez un serveur local avec la commande suivante :
   ```bash
   python -m http.server 8000
   ```
3. Copiez-collez l'adresse suivante ou "ctrl + clic gauche" dessus pour ouvrir une fenêtre.
   http://localhost:8000/syst%C3%A8me_solaire_V2.1.html

## Navigation dans la simulation :
- flèche gauche : gauche
- flèche droite : droite
- flèche haut : avancer
- flèche bas : reculer
- "p" : monter
- "m" : descendre

Vous pouvez ouvrir ou masquer le panneau supérieur avec les boutons + et -, et indiquer une date.
La simulation affichera la configuration du système solaire à cette date et mettra l'animation en pause.
Vous pouvez choisir la vitesse de défilement du temps avec les boutons fléchés en bas.
Le bouton de réinitialisation vous ramène à la configuration des planètes en temps réel, à la vitesse réelle.
